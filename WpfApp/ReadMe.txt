Это WPF-приложение, использующее библиотеку pgDataAccess.dll для доступа к базе данных PostgreSQL.
--------------------------------------------------------------------------------------------------
Требования к приложению - см. папку Docs. Папка Docs/imports - тестовые данные для импорта в базу.
Папка images содержит графические ресурсы для WPF-приложения.
